# 快速启动指南 - 实时监控功能

## 🚀 快速开始

### 1. 启动后端服务
```bash
cd C:\Users\18037\Desktop\ict\ictIPLDS\backend
mvn spring-boot:run
```

等待后端启动完成，看到 "Started PowerInspectionApplication" 提示。

### 2. 启动前端服务
```bash
cd C:\Users\18037\Desktop\ict\ictIPLDS\frontend
npm run dev
```

### 3. 访问应用
打开浏览器访问：`http://localhost:5173`

### 4. 查看实时监控
- 在仪表板页面，找到"实时监控"面板
- 点击不同的无人机按钮切换视角
- UAV-01 会播放视频文件
- UAV-02 显示预留接口提示

## 🔧 故障排查

### 问题：视频无法加载
**症状**：点击 UAV-01 后视频不播放

**解决方案**：
1. 检查后端是否运行：
   ```bash
   curl http://localhost:8081/api/video/info/0f84fa38c795c5dc0f612ca789e306eb_raw.mp4
   ```
2. 检查视频文件是否存在：
   ```bash
   ls C:\Users\18037\Desktop\ict\ictIPLDS\docs\0f84fa38c795c5dc0f612ca789e306eb_raw.mp4
   ```
3. 查看浏览器控制台错误信息（F12）

### 问题：无人机列表为空
**症状**：实时监控面板显示"加载中..."

**解决方案**：
1. 检查后端 API：
   ```bash
   curl http://localhost:8081/api/uav
   ```
2. 检查数据库中是否有 UAV 数据
3. 查看后端控制台日志

### 问题：CORS 错误
**症状**：浏览器控制台显示 CORS 错误

**解决方案**：
1. 确保 VideoController 有 `@CrossOrigin(origins = "*")` 注解
2. 重启后端服务
3. 清除浏览器缓存
